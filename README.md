## Setup & Usage

The following will clone the repo, install dependencies, and run the project. Go to the local IP address output in the console in a web browser to interact with the chatbot.
```
git clone https://github.com/UNHM-TEAM-PROJECT/Fall2025-Team-Goopy.git
cd Fall2025-Team-Goopy
pip install -r requirements.txt
python3 main.py
```
